# Мій перший Telegram-бот на Go

Простий бот, який відповідає на команди та повідомлення.

## Команди
- `/start` - Привітальне повідомлення
- `/help` - Список команд
- `/info` - Інформація про бота
- *Будь-який текст* - Бот повторить його
